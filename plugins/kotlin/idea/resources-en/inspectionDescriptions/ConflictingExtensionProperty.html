Reports extension properties that conflict with synthetic ones automatically produced from Java get or set methods.

Such properties should be either removed or renamed to avoid breaking code by future changes in the compiler.

A quick-fix is suggested to delete an extention property.

Example:


  val File.name: String
      get() = getName()

A quick-fix is suggested to add the @Deprecated annotation:


  @Deprecated("Is replaced with automatic synthetic extension", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
