Reports unlabeled return expression inside inline lambda.

Unlabeled return expressions in lambdas can be confusing because it might be unclear to which scope belongs particular return.

'Change to return@â€¦' quick-fix can be used to amend the code automatically.

Example:


fun test(list: List<Int>) {
    list.forEach {
        // This return expression returns from the function test
        // One can change it to return@forEach to change the scope
        if (it == 10) return
    }
}

After the quick-fix is applied:


fun test(list: List<Int>) {
    list.forEach {
        if (it == 10) return@test
    }
}
